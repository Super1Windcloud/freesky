# Nuxt Minimal Starter

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.


## Docker-Compose 快速开始
- 安装 Docker 
- 安装 Docker Compose 
- 克隆项目到本地
- 进入项目目录 `pnpm run docker` 
 

## 源码构建

#### 使用前提
- Node.js 22+ 
- pnpm 10+ 
- postgreSQL 17+
- ffmpeg 

```bash
pnpm run deploy_server 
pnpm run deploy_client
```
